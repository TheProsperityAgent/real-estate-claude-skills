# Real Estate Claude Skills

**Open-source Claude skills for real estate agents.** Built by [Al & Victoria Pinder](https://theprosperityagent.com), ICON agents at eXp Realty in Greenville, NC.

Every skill is **Fair Housing compliant**, **TCPA-aware**, and **MLS-rule respecting** by default. We never knock a town, builder, neighborhood, or competing platform.

## Quick start

1. Install Claude Code: https://claude.ai/download
2. Open Claude Code
3. Browse `/skills/` here, copy the skill content into a Claude conversation, or clone this repo locally

## Compliance baked in

Every skill in this repo is built to be:

- **Fair Housing compliant** — no school references, no demographic targeting, no language that steers buyers by perception of community
- **TCPA-aware** — never produces unsolicited outreach content; only owned-channel content
- **MLS-rule respecting** — no "must-see", "won't last", or scarcity language that hurts MLS data quality
- **Always positive** — never knocks a town, builder, neighborhood, competing platform, or competing agent

The `fair-housing-overlay` skill (E2) is the foundation. Every other skill in this repo references it. When TCPA/FH rules change, you update one place.

## Why we built this

Real estate agents do the same 30 workflows every week — listing descriptions, offer replies, lead follow-up, social posts, CMAs, presentations, closing graphics. Most "AI for realtors" tools either ignore Fair Housing entirely or bury it in fine print.

We're realtors. We want a tool stack we can defend in front of the broker. So we built one and made it free.
